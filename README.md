# 8-bit-computer-sap-architecture

## Introduction

SAP-8 is an FPGA-based 8-bit computer inspired by the Simple-As-Possible (SAP) architecture and implemented in Verilog HDL. It demonstrates the fundamental concepts of computer organization, including instruction fetch, decode, and execution.

The design consists of modular components such as the Program Counter (PC), Memory Address Register (MAR), Instruction Register (IR), ALU, registers, Control Unit, RAM, and Output Register, which communicate through a common bus system.

This project provides a simple and extensible platform for learning digital design, computer architecture, and FPGA-based processor implementation.
